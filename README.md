# 姿勢分析ツール

整骨院向けの簡易姿勢分析Webアプリ。患者の前後左右の写真から骨格を自動検出し、左右差や前後傾を数値化、AIによる所見コメントを生成します。

## 必要なもの

- 最新のブラウザ（Chrome / Edge / Safari の最新版）
- インターネット接続（MediaPipeモデルのダウンロードと、AI所見生成API呼び出し用）
- お好みのAI APIキー（Gemini / OpenAI / Anthropic のいずれか）
  - **Gemini**：[Google AI Studio](https://aistudio.google.com/) で無料作成可
  - **OpenAI**：platform.openai.com
  - **Anthropic**：console.anthropic.com
  - 「使用しない」を選べばAIなしで計測値だけ表示も可能

## 使い方

### 1. アプリを起動

このフォルダの `index.html` をブラウザで直接開きます。
（または簡易サーバーを立てる場合は `npx serve .` 等でも可）

### 2. 初回設定

ヘッダー右上の「設定」を開き、以下を入力：
- AIプロバイダー：Gemini / OpenAI / Anthropic / 使用しない
- モデル名：プロバイダーを選ぶとデフォルトが入ります（変更可）
- APIキー：取得したキーを入力（端末ローカルの localStorage に保存）

### 3. 患者情報入力（任意）

- 患者氏名・撮影日を入力（端末内のみで保持、サーバ送信なし）
- **AI所見を使う場合は「同意取得済み」のチェックが必要**

### 4. 写真アップロード

正面・背面・左側面・右側面の4枠から、撮影した写真をアップロード。
それぞれ自動で骨格線・基準線（青：肩、橙：骨盤）が描画され、計測値が表示されます。

### 5. AI所見の生成

「AI所見を生成」ボタンで、計測値をAIに送信し、所見・身体への影響・セルフケア提案を生成します。

### 6. レポート印刷

「レポート印刷」でブラウザ印刷ダイアログが開きます。
A4で2〜3枚に収まるレイアウトで、患者にお渡しできます。

## 撮影のコツ

- 患者から **2〜3メートル離れて、カメラの高さを腰の高さ** に
- 全身（頭〜足首）が画面に収まるように
- 背景はできるだけシンプルに、コントラストを出す
- 服装はシルエットがわかる軽装が望ましい
- 4方向すべて、患者の立ち位置を変えずに自分が回り込んで撮影すると比較しやすい

## 計測している項目

| 視点 | 項目 |
|------|------|
| 正面/背面 | 肩の傾き、骨盤の傾き、頭部の傾き、上半身の左右シフト |
| 左/右側面 | 頭部前方位 (FHP)、肩の前方変位、体幹の前後傾、膝の角度 |

数値はすべて MediaPipe Pose Landmarker（Google製、ブラウザ完結のオンデバイスAI）で検出した33点の骨格座標から計算しています。

## プライバシーとセキュリティ

- 写真は **ブラウザ内のみで処理** され、骨格検出に外部送信は一切しません
- AI所見生成時は **計測値の数値のみ** をAPIに送信します（写真は送りません）
- それでも患者個人を特定可能な情報になり得るため、**事前同意の取得を必須**としています
- APIキーは端末の localStorage に保存されます。**共用端末では使用後にキーをクリア**してください
- 本格運用では、ブラウザから直接APIを叩かず、自院サーバーや Cloudflare Workers 等の中継サーバーを挟んでキーを秘匿することを推奨します

## ファイル構成

```
katsu姿勢分析/
├── index.html              # アプリのエントリ
├── styles.css              # スタイル（印刷用CSS含む）
├── README.md               # このファイル
└── src/
    ├── main.js             # 全体フロー制御
    ├── pose/
    │   ├── detector.js     # MediaPipe Pose初期化と推論
    │   └── angles.js       # ランドマーク→角度計算
    ├── ui/
    │   ├── upload.js       # 写真アップロードUI
    │   ├── overlay.js      # Canvas骨格描画と計測値表示
    │   └── report.js       # 所見レンダリングと印刷
    └── ai/
        └── gemini.js       # AIプロバイダー抽象（Gemini/OpenAI/Anthropic）
```

## カスタマイズ

- **計測項目を増やす**：`src/pose/angles.js` の `computeMetrics` に追加
- **所見プロンプト変更**：`src/ai/gemini.js` の `SYSTEM_PROMPT`
- **印刷レイアウト調整**：`styles.css` の `@media print` セクション
- **モデル変更**：`src/pose/detector.js` の `MODEL_URL`（lite / full / heavy の3種類）
